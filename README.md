# html2pdf

Google Chrome Command Line Tool to convert html to pdf.


# Prerequisite

 - Linux
 - Google Chrome

## Examples

URL:

    ./html2pdf https://www.google.com/ google.pdf

Local File:
	
    ./html2pdf file:///home/**YOUR_USERNAME**/www/html2pdf/page.html page.pdf
